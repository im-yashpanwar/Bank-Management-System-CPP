# 🏦 Bank Management System (C++)

A console-based Bank Management System developed in **C++** using **Object-Oriented Programming (OOP)**. This project is part of my learning journey as I build real-world programming skills and strengthen my understanding of software development.

---

# 📖 About the Project

I started this project because I wanted to move beyond small practice programs and build something that feels like a real application.

Instead of creating a simple banking project with only deposit and withdrawal features, I challenged myself to design a system that can gradually evolve into a complete banking simulation.

The goal is not just to finish the project, but to improve it continuously while learning better programming practices, problem-solving, and software design.

# 💡 Biggest Challenge

One challenge I noticed while building this project is that every banking operation asks for the Account Number and PIN again.

Current flow:

Deposit
→ Enter Account Number
→ Enter PIN

Withdraw
→ Enter Account Number
→ Enter PIN

Transfer
→ Enter Account Number
→ Enter PIN

This works, but it doesn't feel like a real banking application.

---

# 🔄 Version 2 Improvements

Version 2 will introduce a proper login system.

Instead of repeatedly entering account details, users will:

Welcome Screen

- Create Account
- Login
- Exit

After Login:

- Deposit
- Withdraw
- Transfer
- Upgrade Card
- Loan
- Display Account
- Logout

This will make the application closer to a real banking system while improving the overall user experience.
